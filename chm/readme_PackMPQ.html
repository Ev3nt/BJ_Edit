PackMPQ 说明文档


简介：
PackMPQ能将地图直接打包成mod

在mod制作的最后阶段
你只需选定用BJ Editor制作好了用于mod的Blizzard.j，连同一些附加数据，便可以直接打包成用于mod的mpq。
使用说明：
  1. 使用BJ Editor生成合并了地图脚本的Blizzard.j，具体说明请参照BJ Editor说明文档

  2. 选取生成好的Blizzard.j路径，如果你生成的BJ并不在Output文件夹里。

  3. 将不在地图的资源放入'Additional_Resources'文件夹中，例如：游戏背景，logo，主界面UI...

  4. 按下'Pack!'输入要保存的mpq文件名，稍等片刻，就可以用这个mpq连同主程序进行游戏了。
